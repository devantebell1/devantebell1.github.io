Work

I created this static photography site using just HTML and CSS. This project was important to me because I created this around the same time as the George Floyd tragedy and Minneapolis riots. Being a person of color and a Minnesota native I had an enormous amount of different emotions during this time. I thought it would be creative and unique to document these events through photographs on a website.

I created a simple to do web app using javascript, css, and html. In this project I focused more on basic javascript functions by creating cool animations when you delete an item from the list or filtering from completed and non completed tasks.

I created a weather app using css, html, and vanilla javascript. In this project I pulled out timezone, temperature, and temperature description information from DarkSky API to display local weather info. I then used SkyCons to display a animated svg icon depending on the tempature description. You are also able to switch between Farenheit and Celsius!
